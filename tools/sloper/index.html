Sloper Tool

原型作成ツール

KATAPATA本体を開いて、サイズ入力から原型作成を始めます。初めて使う場合は、先にガイドを見ると流れがつかみやすいです。

Main Tool

KATAPATAを開く

ブラウザ上で原型を作成します。入力内容を確認しながら進め、必要に応じてPDF出力へ進んでください。

原型づくりをはじめるはじめてガイドを見る

作業前のメモ

入力単位

cm / inch の切替に対応しています。内部計算はcmを基準にしています。

PDF出力

無料出力は縮小サンプルPDFです。有料出力では、通常サイズPDFとA4分割印刷PDFをセットで使えます。

身頃・袖の原型

文化式婦人原型をもとに、身頃と袖の原型を作成します。

スカート・パンツ原型

スカート原型とパンツ原型は、それぞれ別の出力枠として整理しています。
