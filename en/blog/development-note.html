Dev Note

Why KATAPATA is becoming a full website

KATAPATA began as a tool for entering measurements, drafting slopers, and exporting PDFs. But as it has moved closer to release, there are more and more things that cannot be explained well inside the tool screen alone.

To keep the tool and explanations separate

What is a sloper? What is the reduced-size sample PDF? What should be checked before printing? If all of those explanations were placed inside the tool screen, the main interface would become harder to read.

So the tool itself stays as simple as possible, while longer explanations are placed in guide pages and blog articles.

To help first-time users feel less lost

KATAPATA is meant for people who already enjoy sewing, as well as people who are trying sloper drafting for the first time.

Instead of showing only an input screen, the website gives users places to understand what the tool can do, what can be checked for free, and how the PDF output works.

To make updates easier later

Pricing, FAQ, printing notes, and usage guides may be revised little by little after release. Keeping them as separate pages makes it easier to update only the parts that need attention.

It also makes it easier to organize Japanese and English pages separately, so the information can be clearer for users outside Japan too.

To make the site feel like KATAPATA

KATAPATA should not be only a list of functions. I want the site to feel calm and easy to read, so that the anxiety around sloper drafting becomes a little lighter.

By adding small notes and articles over time, this blog can become a place that answers questions before and after using KATAPATA.
